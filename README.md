# React JS Workspace

This folder contains multiple React projects built while learning from the CodeWithHarry React JS course.

## Projects

- `textutils` - Text utility app with theme toggle and routing
- `newsapp` - News app project
- `reduxapp` - Basic Redux practice app for deposit and withdraw flow
- `iNotebook` - Notes app with separate frontend and backend

## Tech Stack

- React
- Vite
- Bootstrap
- Redux and React Redux
- Node.js and npm

## Getting Started

Move into any project folder and run:

```bash
npm install
npm run dev
```

Example:

```bash
cd reduxapp
npm install
npm run dev
```

## Notes

- Each project has its own `package.json`
- `iNotebook` contains both frontend and backend parts
- Project-specific details can be added inside each app's own `README.md`
